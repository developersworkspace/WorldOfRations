Features

Formulates on a true “Least-Cost” procedure

Our program formulates rations on a least-cost linear programming procedure. The user merely has to select from the extensive list of feedstuffs the feeds available to him and submit the cost of the selected ingredients.

Indicates the composition and nutrient status

The program provides (together with the ration composition) the ration’s nutrient status, by listing the nutrients that are adequate, deficient or in excess. In addition, you receive a customized micro-nutrient supplement mix which, when added to your 1000 kg mix, will rectify any deficiencies found in the initial macro formulation.
